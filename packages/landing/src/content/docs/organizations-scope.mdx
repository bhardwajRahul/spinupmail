## Scope model

Session-cookie requests use the active org from the user session. API key requests must pass `X-Org-Id`.

<Callout tone="warning" title="Missing X-Org-Id">
API key requests without `X-Org-Id` are rejected because org scope cannot be inferred safely.
</Callout>
