## List and detail endpoints

<Endpoint method="GET" path="/api/emails">
List emails for an address in current org scope.
</Endpoint>

<Endpoint method="GET" path="/api/emails/:id">
Get full message detail with body and attachment metadata.
</Endpoint>

```bash
curl "/api/emails?addressId=addr_123" \
  -H "X-API-Key: spin_..." \
  -H "X-Org-Id: org_abc123"
```

## Downloads

<Endpoint method="GET" path="/api/emails/:id/raw">
Download raw MIME when raw storage is enabled.
</Endpoint>

<Endpoint method="GET" path="/api/emails/:id/attachments/:attachmentId">
Stream a specific attachment binary.
</Endpoint>

```bash
curl -L "/api/emails/mail_123/attachments/att_987" \
  -H "X-API-Key: spin_..." \
  -H "X-Org-Id: org_abc123" \
  --output attachment.bin
```

<Endpoint method="DELETE" path="/api/emails/:id">
Delete an email row and linked attachment references.
</Endpoint>
