## Worker config

```toml
[vars]
EMAIL_DOMAINS = "spinupmail.com,spinuptestdomain.com"
```

## Routing rule checklist

1. Enable Email Routing for each domain.
2. Add destination address for each domain.
3. Create `*@domain` -> Worker routing rule for each domain.

## UI behavior

When multiple domains are configured, address creation displays a domain selector. With a single domain, it is preselected automatically.
