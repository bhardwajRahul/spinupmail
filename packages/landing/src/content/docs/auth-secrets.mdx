## Required Worker secrets

```bash
pnpm exec wrangler secret put BETTER_AUTH_SECRET
pnpm exec wrangler secret put BETTER_AUTH_BASE_URL
pnpm exec wrangler secret put GOOGLE_CLIENT_ID
pnpm exec wrangler secret put GOOGLE_CLIENT_SECRET
pnpm exec wrangler secret put RESEND_API_KEY
pnpm exec wrangler secret put TURNSTILE_SECRET_KEY
```

## Set Better Auth Base URL

Use your deployed auth endpoint, for example `https://your-domain.com/api/auth`.

<Callout tone="warning" title="Trusted origin alignment">
`CORS_ORIGIN` must include your frontend origins, or callback validation will fail.
</Callout>

## Google OAuth callback setup

1. Configure OAuth consent screen in Google Cloud.
2. Add authorized origins for local and production frontend URLs.
3. Add callback URL(s): `/api/auth/callback/google`.
4. Store credentials with `wrangler secret put`.
