## Prerequisites

- A Cloudflare account with a domain on Cloudflare nameservers
- Cloudflare Email Routing enabled for the domain
- `pnpm` installed locally

## Install and run

From the repository root, install dependencies and start backend plus frontend.

```bash
pnpm install
pnpm -C packages/backend dev
pnpm -C packages/frontend dev
```

<Callout tone="info" title="Local API routing">
The frontend dev server proxies `/api/*` requests to the backend worker in dev.
</Callout>

## First successful flow

1. Sign up or sign in.
2. Create an organization.
3. Create a temporary email address.
4. Send an email to that address.
5. Confirm it appears in mailbox view.

### First API check

Verify organization scope by calling an API endpoint with `X-Org-Id`.

```bash
curl "/api/email-addresses" \
  -H "X-API-Key: spin_..." \
  -H "X-Org-Id: org_abc123"
```
